 ## 🧑‍💻 Project-Monty Hall Problem
#### 📋 프로젝트 소개

  
이 프로젝트는 몬티홀 문제(Monty Hall Problem)를 시뮬레이션하고, 두 가지 전략(선택 유지 vs 선택 변경)의 승리 확률을 비교하는 프로그램입니다. 사용자가 실험 횟수를 입력하면 시뮬레이션을 실행하고, 결과를 막대그래프로 시각화합니다. 이를 통해 이론적 확률과 실제 결과의 일치 여부를 확인할 수 있습니다. 



#### 💡 프로젝트 목표 


1. **몬티홀 문제의 검증**  
   - "선택 유지"와 "선택 변경" 전략 간의 승리 확률을 시뮬레이션으로 증명.  
   - 조건부 확률을 이용한 실험 결과와 이론적 확률(33.3% vs 66.7%)의 일치 여부 확인.  

2. **사용자 친화적 인터페이스 제공**  
   - 반복 횟수를 사용자로부터 입력받아 원하는 규모의 실험을 실행.  
   - 결과를 시각화하여 데이터를 직관적으로 이해할 수 있도록 지원.  

3. **프로그래밍 학습 및 응용**  
   - 확률 문제를 코드로 구현하며 Python 프로그래밍 스킬을 향상.  
   - `random`과 `matplotlib` 라이브러리를 활용한 시뮬레이션 및 데이터 시각화 기법 연습.



  
#### 1️⃣ 환경 설정  
- Python 3.x가 설치된 환경에서 실행 가능합니다.  
- 필요한 라이브러리를 설치합니다:  
  ```bash
  pip install matplotlib
</summary>


#### 🛠️ 사용 기술  

- **언어**: <img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=Python&logoColor=white" width="7%"/>
- **라이브러리**:  
  - `random`: 자동차 위치 및 플레이어 선택에 랜덤성을 제공.  
  - `matplotlib`: 실험 결과를 막대그래프로 시각화.  

#### 📊 결과 시각화

실험 결과는 아래와 같은 막대그래프로 나타납니다:

![스크린샷 2024-11-22 173933](https://github.com/user-attachments/assets/d9a75fde-2406-428a-a1b3-719c690204f7)

*예시 그래프: 선택을 바꾼 경우(Change)와 바꾸지 않은 경우(No Change)의 승리 확률 비교.*

##### 예상 결과  
- **선택을 바꾼 경우**(Change)의 승리 확률은 약 66.7%로, **선택을 유지한 경우**(No Change)는 약 33.3%의 승리 확률을 보입니다.
- 시뮬레이션을 통해 이 이론적 확률을 실험적으로 검증할 수 있습니다.
